Remember

More

Remember was a project that focused on designing for automation in the smart home. We did a prototype with tangible interactions that were meant to manage automation in the smart home.

The concept consisted of a screen connected to all smart applications in a user's home, as well as a special door handle. When the door handle was interacted with in a special way, the screen would display all connected smart devices that were not turned off, and the user could control them from the screen on the door.

For the purpose of the project, we instead created a physical prototype that sounded an alarm from the forgotten devices.

Cardholder

More

During the Visual Communications course we got an assignment to create an App that used wearable technologies.
I decided to create a concept that took all of a user's membership cards and stored them all in one place, easily accessible through the user's phone. This allows for the user to not worry about carrying around all of their cards in their wallet, since most of them are commonly not used every day.

The first step was to create the concept and to sketch it out. After that, I created a storyboard to display the flow of the application, and what the thought area of use is. The last step of this project was to create interface mock-ups with Illustrator and Photoshop.
